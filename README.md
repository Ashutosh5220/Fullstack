# 🚀 Full Stack Task Manager (MERN)

A full-stack Task Management system with authentication, JWT security, and CRUD operations. Built as part of an internship assignment.

---

## 📌 Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- bcrypt

### Frontend
- React (Vite)
- JavaScript
- Fetch API

---

## ✨ Features

### 🔐 Authentication
- User Registration
- User Login
- Password hashing (bcrypt)
- JWT token authentication

### 📋 Task Management (CRUD)
- Create Task
- Read Tasks
- Update Task
- Delete Task

### 🔒 Security
- JWT protected routes
- Password encryption
- User-specific task access
